# Public Assets

## Logo

Place your Upixora logo image here as `logo.png` or `logo.svg`.

The logo should be:
- PNG or SVG format
- Recommended size: 40x40px to 200x200px
- Transparent background (optional but recommended)

The logo will be displayed in:
- Navigation bar (with "Upixora" text)
- Footer (with "Upixora" text)

If you don't have the logo file yet, the application will show a broken image placeholder until you add it.
